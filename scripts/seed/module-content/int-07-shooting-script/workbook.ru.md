# Режиссёрский сценарий
**Moirai — сохрани этот документ**

---

## КРАТКО ПО ЛЕКЦИИ

Литературный сценарий описывает, что происходит. Режиссёрский сценарий описывает, как это будет снято.

Шесть элементов каждого плана: общая информация / структура плана / действие и состояние персонажа / локация и свет / звук и монтаж / технические заметки.

Два практических применения: страховка по continuity (защита от ошибок) и бюджетный инструмент (снимаем только то, что нужно).

---

## КЛЮЧЕВЫЕ ПОНЯТИЯ

**Режиссёрский сценарий** — рабочая инструкция для всей съёмочной группы. Не для чтения — для использования на площадке.

**Крупности планов:** WS (общий) / MS (средний) / CU (крупный) / ECU (экстра-крупный) / POV (с точки зрения персонажа) / Insert (вставка)

**Движение камеры:** статика / panoramика (Pan) / наклон (Tilt) / тележка (Dolly) / с рук (Handheld) / Steadicam / кран (Crane)

## РЕАЛЬНЫЙ ПРИМЕР — ЗАПОЛНЕННЫЙ РЕЖИССЁРСКИЙ СЦЕНАРИЙ

Стандартный формат, используемый в профессиональных продакшнах (StudioBinder / индустриальный стандарт):

```
СЦЕНА 7 · ИНТ. КУХНЯ · УТРО
Цель сцены: Анна находит записку. Понимаем, что кто-то ушёл.
```

| ПЛАН | КРУПН. | ДВИЖ. | РЕКВИЗИТ | ДЛИТ. | ДЕЙСТВИЕ + СОСТОЯНИЕ ПЕРСОНАЖА |
|---|---|---|---|---|---|
| 7.1 | WS | | чашка, кофе | ~6 сек | Анна входит слева. Идёт к кофеварке. Видит записку на столе. Останавливается. |
| 7.2 | MS | | записка | ~10 сек | Анна берёт записку. Читает. Лица не видим — только руки и макушка. |
| 7.3 | CU | Статика | записка | ~8 сек | Её лицо. Дочитывает. Аккуратно кладёт записку обратно. |
| 7.4 | Деталь | Статика | | ~4 сек | Записка на столе. Текст не разобрать — видно, что коротко. |

```
СЦЕНА 7 РЕКВИЗИТ: записка × 3, кружка × 2
КОСТЮМ: пижама → сфотографирована [дата]
CONTINUITY: записка лежит ближе к левому краю стола
```

## ГЛОССАРИЙ

**EWS** — Extreme Wide Shot (сверх-общий план)

**WS** — Wide Shot (общий план)

**FS** — Full Shot (план в полный рост)

**MWS** — Medium Wide Shot (средне-общий план)

**MS** — Medium Shot (средний план)

**MC** — Medium Close-Up (средне-крупный план)

**CU** — Close-Up (крупный план)

**ECU** — Extreme Close-Up (экстра-крупный план)

**Деталь** — крупный план объекта, изолированный от основного действия сцены.

**ИНТ / НАТ** — интерьер / натура. Указывается в шапке каждой сцены.

**Статика / Pan / Tilt / Dolly / Handheld / Steadicam** — типы движения камеры.

**Continuity** — единообразие костюмов, реквизита и положений между съёмочными днями.

**Shot list** — упрощённая версия: нумерованный список планов без подробных описаний.

**Insert (вставка)** — крупный план объекта или детали, изолированный от основного действия сцены.

**Room tone** — фоновый звук локации без диалогов и действий. Записывается отдельно для монтажа.

**Оператор-постановщик (DP)** — отвечает за изображение: свет, композицию, оптику.
