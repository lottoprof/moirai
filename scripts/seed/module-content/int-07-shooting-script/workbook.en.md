# The Director's Script

## LECTURE SUMMARY

The screenplay describes what happens. The director's script describes how it will be filmed.

Six elements of every shot: general information / shot structure / action and character / location and light / sound and edit / technical notes.

Two practical uses: continuity insurance (protection from mistakes) and budget tool (shoot only what you need).

---

## KEY CONCEPTS

**Director's script** — a working instruction for the entire crew. Not for reading — for use on set.

**Shot types:** WS (wide) / MS (medium) / CU (close-up) / ECU (extreme close-up) / POV / Insert

**Camera movement:** Static / Pan / Tilt / Dolly / Handheld / Steadicam / Crane

---

## REAL EXAMPLE — FILLED-IN DIRECTOR'S SCRIPT

Standard format used in professional productions (StudioBinder / industry standard):

```
SCENE 7 · INT. KITCHEN · MORNING
Scene goal: Anna finds the note. We understand someone has left.
```

| SHOT | TYPE | MOVE | PROPS | DUR. | ACTION + CHARACTER STATE |
|---|---|---|---|---|---|
| 7.1 | WS | | cup, coffee | ~6 sec | Anna enters from left. Walks to coffee maker. Sees note on table. Stops. |
| 7.2 | MS | | note | ~10 sec | Anna picks up the note. Reads it. We don't see her face — only hands and top of head. |
| 7.3 | CU | Static | note | ~8 sec | Her face. She finishes reading. Puts the note down carefully. |
| 7.4 | Detail | Static | | ~4 sec | The note on the table. We can't read the words — we see it's short. |

```
SCENE 7 PROPS: note × 3, mug × 2
COSTUME: pajamas → photographed [date]
CONTINUITY: note lies closer to the left edge of the table
```

## GLOSSARY

**EWS** — Extreme Wide Shot

**WS** — Wide Shot

**FS** — Full Shot

**MWS** — Medium Wide Shot

**MS** — Medium Shot

**MC** — Medium Close-Up

**CU** — Close-Up

**ECU** — Extreme Close-Up

**Detail** — close-up of an object, isolated from the scene's main action

**INT / EXT** — Interior / Exterior. Listed in the header of every scene.

**Static / Pan / Tilt / Dolly / Handheld / Steadicam** — camera movement types.

**Continuity** — consistency of costumes, props, and positions across shooting days.

**Shot List** — simplified version: a numbered list of shots without detailed descriptions.

**Insert** — a close-up of an object or detail, isolated from the scene's main action.

**Room tone** — the ambient sound of a location with no dialogue or action. Recorded separately for editing.

**DP (Director of Photography)** — the cinematographer. Responsible for the image: light, composition, lenses.
